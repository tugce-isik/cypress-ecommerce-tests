# Cypress E-Commerce UI Tests

In diesem Projekt sind automatisierte Cypress-Tests für die Demo-Webseite automationexercise.com enthalten.

### ✔️ Test Features:
- Anmelden
- Produktsuche
- In den Warenkorb

### 🔧 Technologies:
- Cypress v20.18.1
- JavaScript (ES6)

### ▶️ Run Tests:
```bash
npm install
npx cypress open
